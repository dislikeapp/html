About OmniEngine

OmniEngine is a JavaScript API for making 3D web browser games using WebGL. OmniEngine is currently beta, but we would certainly appreciate users and feedback. Our library is available on the GitHubs or you can just link to it at http://omniserver.no-ip.biz/main/projects/oe-js/OmniEngine/bin/OmniEngine.js
